互联网及其影响

互联网的起源、应用特征、发展趋势与潜在风险

互联网的起源

  • 起源：与阿帕网（ARPANET）有关，中国于1994年接入互联网
  • 互联网应用及功能：搜索引擎、即时通信、在线支付等
  • 互联网应用特征：操作简单、交互性强、图文声像并茂
  • "互联网+"："互联网+各传统行业"，推动传统行业转型升级

互联网潜在风险

  • 信息安全与隐私保护问题
  • 青少年网络成瘾与心理健康影响
  • 信息过载与虚假信息传播
  • 数字鸿沟与社会不平等加剧
  • 网络暴力与不良文化影响

搜索技巧对比

精确匹配 vs 模糊搜索

❌ 模糊搜索
搜索：人工智能

结果：包含"人工"和"智能"两个词的所有网页
数量：约1亿条结果

✅ 精确匹配
搜索："人工智能"

结果：完整包含"人工智能"这个词组的网页
数量：约1000万条结果

提示：使用引号可以让搜索引擎精确匹配你要找的完整词组，大幅减少无关结果！

HTML 网页结构

<!DOCTYPE html> <html> <head> <title>我的第一个网页</title> </head> <body style="background-color: #FFFFFF; opacity: 0.8"> <h1>你好，世界！</h1> <p>这是一个段落。</p> </body> </html>
代码解析：
• <!DOCTYPE html> — 声明文档类型
• <html> — 根标签
• <head> — 头部信息（标题、样式等）
• <body> — 网页主体内容
• opacity: 0.8 — 设置80%不透明度

Unit Four

互联网基本原理

网络分类、IP地址、域名系统与网络协议

网络分类与连接

网络分类

  • LAN — 局域网（学校、办公室）
  • WAN — 广域网（互联网）

网络连接设备

路由器：路由选择、数据转发

网络连接方式

  • 有线：双绞线、光纤
  • 无线：Wi-Fi、蓝牙
常用命令：
• ipconfig/all — 查看IP配置
• ping — 检测连通性

IP 地址

定义

标识互联网中每台主机和路由器，具有唯一性

组成

网络号 + 主机号

地址分类

公有地址（全球唯一）
私有地址（内部使用）

类型 位数 示例
IPv4 32位 192.168.0.111
IPv6 128位 更多地址空间
私有地址范围：
10.0.0.0 - 10.255.255.255
172.16.0.0 - 172.31.255.255
192.168.0.0 - 192.168.255.255

域名与网络协议

域名系统 DNS

  • 将域名转换为IP地址
  • URL：统一资源定位符
DNS解析示例
www.baidu.com ↓ DNS解析 110.242.68.66
电子邮件格式

用户名@邮件服务器域名

如：student@163.com

网络协议

  • HTTP：超文本传输协议
  • SMTP：发送邮件
  • POP3：接收邮件
  • IMAP：同步邮件
  • TCP/IP：基础通信协议
协议类比：
HTTP = 快递单（告诉送什么）
TCP = 物流系统（确保送达）
IP = 地址（送到哪里）

    网络风险识别、身份验证与数据加密

    常见风险与防范措施

    ⚠️ 常见风险

    • 网络诈骗
    • 恶意软件
    • 信息泄露
    • 网络欺凌
    • 生成式AI新风险

    🛡️ 防范措施

    • 安装配置防火墙
    • 定期更新系统
    • 设置复杂密码
    • 防范恶意软件
    • 提高安全意识

    身份验证三大因素

    🧠
    知识因子
    密码、PIN码、图案锁
    📱
    持有因子
    手机、U盾、智能卡
    👤
    生物因子
    指纹、人脸、虹膜
    数据加密：将明文与密钥结合，通过特定算法转换为密文，分为对称加密和非对称加密
    安全对比

    强密码 vs 弱密码

    ❌ 弱密码
    123456
    password
    a1b2c3d4
    20000101

    ⚠️ 易被破解，包含连续数字、常见单词或纯数字

    ✅ 强密码
    Tr@in!2024
    K#9mP$xL2q
    MyH0use#2024
    B!ue$Ky777

    ✓ 包含大小写字母、数字、特殊字符，长度8位以上

    ⚠️ 网络诈骗识别

    • "冒充老师/领导"收费 → 核实身份
    • "中奖了，先交手续费" → 虚假诱饵
    • "点击链接领取奖品" → 钓鱼网站
    • "账号异常，需验证" → 官方核实
