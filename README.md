# 信息技术复习提纲 - Interactive Presentations

Interactive HTML presentations for Grade 7 and Grade 8 ICT exam preparation, deployed via Streamlit.

## 📚 Contents

- **七年级 (Grade 7):** 互联网基础、信息处理、网站制作、网络原理、创新应用、安全防范、人工智能
- **八年级 (Grade 8):** 物联网架构、感知层、网络层、物联应用、安全发展

## 🚀 Local Development

```bash
# Install dependencies
pip install -r requirements.txt

# Run Streamlit app
streamlit run app.py
```

## 🌐 Deployment

This app is deployed on [Streamlit Community Cloud](https://share.streamlit.io/).

### Features

- Interactive quizzes with instant feedback
- Keyboard navigation (↑↓ arrows, Space)
- Responsive design for mobile/tablet
- Inline editing mode (press E)
- 40+ knowledge points covered
- 6-7 units per grade

### Tech Stack

- **Streamlit** - Web framework
- **HTML/CSS/JS** - Presentations
- **clamp()** - Fluid typography

## 📖 Usage

1. Select grade from sidebar
2. Navigate slides with arrow keys or scroll
3. Take quizzes after each unit
4. Press `E` to edit content inline
